1. install the plugin:

vagrant plugin install vagrant-disksize

2. usage:

Vagrant.configure('2') do |config|
  config.vm.box = 'ubuntu/xenial64'
  config.disksize.size = '100GB'
end

3. ssh the virtual machine, run the script ro add disk:

cat add-disk.sh

#!/bin/bash

fdisk /dev/sda << EOF
n





w
EOF

4. reboot

5. write /etc/fastab, and mount -a
